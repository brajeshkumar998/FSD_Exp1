1. Working with HTML Lists

HTML provides several ways to create lists to organize content on your web pages.

HTML Code for Lists

<!-- Ordered List -->
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

<!-- Unordered List -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Definition List -->
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

2. Working with Hyperlinks

The <a> tag defines a hyperlink, which is used to link from one page to another.

HTML Code for Links

<!-- Basic link -->
<a href="https://example.com">Visit Example.com</a>

<!-- Link with target attribute -->
<a href="https://example.com" target="_blank">Open in new tab</a>

<!-- Link to email -->
<a href="mailto:contact@example.com">Send Email</a>

<!-- Link to phone number -->
<a href="tel:+1234567890">Call Us</a>

3. Profile Images with Links

Images can be made clickable by wrapping them with an <a> tag.

HTML Code for Profile Images

<!-- Your profile -->
<div class="profile">
  <a href="your_profile.html">
    <img src="your_image.jpg" alt="Your Name" 
         width="150" height="150" class="profile-img">
  </a>
  <p>Your Name</p>
</div>

<!-- Friend's profile -->
<div class="profile">
  <a href="friend_profile.html">
    <img src="friend_image.jpg" alt="Friend's Name" 
         width="150" height="150" class="profile-img">
  </a>
  <p>Friend's Name</p>
</div>

4. Creating an Image Gallery with Thumbnails

Thumbnails are small versions of images that link to larger versions.

HTML Code for Image Gallery

<div class="gallery">
  <a href="large1.jpg">
    <img src="thumb1.jpg" alt="Image 1" 
         width="100" height="100" class="thumbnail">
  </a>
  <a href="large2.jpg">
    <img src="thumb2.jpg" alt="Image 2" 
         width="100" height="100" class="thumbnail">
  </a>
  <!-- More thumbnails -->
</div>
